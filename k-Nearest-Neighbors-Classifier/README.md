# Machine Learning Algorithms 

This repository serves as a practical codebase demonstrating foundational Machine Learning algorithms implemented entirely from scratch using Python and mathematical libraries. The primary goal is to master the core matrix operations, statistical formulas, and computational mechanics behind complex predictive models without relying on high-level frameworks like Scikit-Learn.

##  Implemented Algorithms

* **`k-Nearest-Neighbors-Classifier/`**
  * A vector-based classification model utilizing Euclidean distance matrices to dynamically isolate nearest neighbors and resolve predictive target states via majority voting.
